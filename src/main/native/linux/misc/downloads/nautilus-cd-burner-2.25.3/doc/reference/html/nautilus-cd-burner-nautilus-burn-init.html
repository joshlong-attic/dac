nautilus_burn_init ()

gboolean            nautilus_burn_init                  (void);

If nautilus-burn is not already initialized, initialize it. This must be called prior to performing any other nautilus-burn operations, and may be called multiple times without error.

Returns :

 TRUE if nautilus-burn is successfully initialized (or was already initialized).

nautilus_burn_initialized ()

gboolean            nautilus_burn_initialized           (void);

Detects if nautilus-burn has already been initialized (nautilus-burn must be initialized prior to using any methods or operations).

Returns :

 TRUE if nautilus-burn has already been initialized.

nautilus_burn_shutdown ()

void                nautilus_burn_shutdown              (void);

Cease all active nautilus-burn operations.
